Din bästa vän som student, här kan du få hjälp med mer än att bara hitta böcker.

Här på Ditt Universitet finns det 4328 olika kurser att läsa. Under kurser kan du söka kurser på kursnamn eller kurskod eller välja ett ämne som intresserar dig i filtreringsfunktionen och få se alla kurser som finns tillgängliga i det valda ämnet. I kurslistan kan du spara kurser som sedan presenteras i ramschemat. I ramschemat kan du se totalt antal högskolepoäng som dom sparadekurserna får tillsammans. Utöver det kan du självklart också ta bort kurser från listan ifall du ångrat ditt val.
